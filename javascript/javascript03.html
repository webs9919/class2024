01. 함수 : 선언적 함수

선언적 함수(Declaration function)는 function 키워드를 사용하여 정의된 함수입니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
